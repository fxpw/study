2. Создать структуру моделей данных с помощью SQLAlchemy 
3. По структуре моделей создать базу данных (база данных sqlite)  
 
4. Написать скрипты для заполнения базы данных данными и выборки данных из базы (Все делаем через ORM), возможны 2 вариант: 1) более простой - использовать просто какие то тестовые данные, сохранить их в базу и после этого сделать выборку этих данных и вывести в терминал. 2) более сложный - использовать реальные данные, которые получил парсер, вместо json - а теперь сохранить эти данные в созданную базу, сделать выборку данных либо в консольном варианте программы, либо вывести на сайт (идеальный вариант)
